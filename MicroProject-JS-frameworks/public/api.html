API Demo: Fetching Car Data

Welcome to the API Demo page! Here, we demonstrate how to retrieve car data and display it dynamically on the webpage. Currently, we are fetching this data from a local JSON file using a callback function to simulate the process of loading data.

Once the "Get Car Data With Async/Await" button is clicked, the page will load car information such as make, model, price, year, and car images into a table format. This data is currently being fetched from a local JSON file, which contains car details of various brands and models.

Currently, the data is being retrieved via a callback function from this static JSON file. However, we plan to enhance the functionality by integrating an external API in the future to fetch real-time data for the cars. This will allow for greater scalability and the inclusion of up-to-date vehicle listings.
